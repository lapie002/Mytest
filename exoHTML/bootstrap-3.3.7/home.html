Bugs communs

  • Bugs communs/fréquents :
    1. Mauvaise orthographe entre 2 fichiers pour inclure un fichier, class, id, fonctions
    2. Travailler sur un fichier dans l'éditeur mais regarder le rendu/résultat sur un autre fichier dans le navigateur
    3. Problème de cache ( Affiche une vieille version de la page html et on voit pas les modifications -> faire Ctrl+F5 plutôt qu'un simple rafraichissement )
    4. Problème de "case sensitive" == majuscule et minuscule , écrire id="Passowrd" dans le html et "password" dans le JS ou CSS == Erreur, voir les conventions ci-dessous et toujours utiliser le camelCase

Conventions

  • Les images dans un repertoire img/
  • Les css dans un repertoire css/
  • Utiliser le CamelCase et plus particulierement le lowerCamelCase Exemple : displayResult
  • Pas d'espace, accents, ponctuation dans le nom des répertoires, fichiers ou variables
  • Pas d'image de plus de 1000pixels ou 1MB sauf si ça sert de background du site
