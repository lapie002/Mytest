Lapierre Bruno - promotion 2017

Semaine 3 : du 31 Janvier au 3 Fevrier.

Initiation au Javascript.

Jour 11

Nous avons été jusqu'à l'activité d'OpenClassroom : Créez un mini-jeu de devinette.
Les variables en JS.
  • String
  • Numerique
  • Boolean
  • Tableau
  • Objet
  • Null
  • Undefined
Une autre partie de la journée, j'ai installé Apache en local grace au tutoriel trouvé sur DigitalOcean et j'ai commencé à decouvrir les lignes de commande en "bash".

Jour 12

Nous avons finis la partie une du tutoriel sur le Javascript apprenez a coder avec javascript.
Et nous avons vu des adresses de sites utiles pour la documentation du JS : EloquentJS
Nous avons un exercice pour devouvrir les méthode applicablent sur les tableaux.
  • .push();
  • .splice();
  • .indexOf();
  • .join();
  • .split();
  • .sort();

Jour 13

Nous avons fait deux exercices sur le javascript grâce à la leçon disponible en ligne sur le DOM CodyMaster et exoDOM-master.
En debut de matiné, nous avons corrigé les exercices de la veille. L'après midi, introduction sur le DOM et les deux exercices mentionnés au dessus.

Jour 14

Nous avons finis la corrections des exercices de la veille, nous avons un exercice pour decouvrir les fonctions en Javascript : index.html, math.js
L'après-midi, nous avons installé la bibliotheque jQuery, puis avec CDN (content distribution network). Une bonne adresse pour les tutoriels CSS

Jour 15

Integration d'un Accodion en jQuery UI : le fichier HTML et le script JS
Un exercice de correction des erreurs "FindErros"HTML et script.js, app.js

Semaine 4 : du 6 Fevrier au 10 Fevrier.

Approdissement en Javascript sur la modification du DOM.

Jour 16

Nous avons commencé la journée par un exercice avec un formulaire : exologin.html , exologin.js, exologin.css.
Dans l’après midi, nous nous sommes concerté sur le futur template pour la promotion avec une sélection et une critique de différents sites web. Chaque personne a donné l'adresse d'un site esthétique et d'un site non ergonomique.

Jour 17

Nous avons fini la lecture du tutoriel openclassrooms sur le Javascript. Et reçu le premier projet : jeu de piste familial à la découverte de Narbonne mythologique.
L’après midi, nous avons travaillé en groupe sur la creation d'un page web (HTML et CSS) pour présenter un lexique sur des technologies web (mots présentés KISS, CSS).

Jour 18

Nous avons commencé l'intergration du site web www.animalwebaction.com
L’après midi, nous avons continué à travailler sur le site animalwebaction et fini avec la correction de l'intergration du site .
