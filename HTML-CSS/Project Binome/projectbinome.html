Kiss :

NON ce n'est pas le célèbre groupe de rock des années 70, kiss est une abreviation pour "Keep It Stupid Simple" ou "Keep It Simple, Stupid.."

Qu'est-ce que ça veut dire ? littéralement, comprenez mot à mot : « garde ça simple, idiot », dans le sens de « ne complique pas les choses »

Ce principe à été un élement clef, et un énorme succès dans le domaine de l'ingénierie logicielle. Un problème courant chez les ingénieurs et développeurs de logiciels aujourd'hui, c'est qu'ils ont tendance à compliquer les problèmes.

En règle générale, lorsqu'un développeur est confronté à un problème, il le décompose en plus petits morceaux qu'il pense comprendre, puis tente de mettre en œuvre la solution dans le code. Je dirais que 8 ou 9 sur 10 développeurs font l'erreur qu'ils ne décomposent pas le problème en assez petit ou compréhensible morceaux. Cela a pour résultat des implémentations très complexes même des problèmes les plus simples, un autre effet secondaire est le code "spagetthi", quelque chose que nous pensons que seul BASIC pourait faire avec ses déclarations goto, mais en Java cela donne des classes de 500 à 1000 lignes de code, dont chaque méthodes a plusieurs centaines de lignes..

Ce désordre de code resulte des cas d'exception à sa solution originale que met en place le dévelopeur pendant qu'il écrit dans son code. Ces cas d'exception auraient été résolu si le développeur avait décomposé le problème plus tôt.

CSS:

Le terme CSS est l'acronyme anglais de Cascading Style Sheets qui peut se traduire par "feuilles de style en cascade". Le CSS est un langage informatique utilisé sur l'internet pour mettre en forme les fichiers HTML ou XML. Ainsi, les feuilles de style, aussi appelé les fichiers CSS, comprennent du code qui permet de gérer le design d'une page en HTML.


Bien que l'HTML puisse être mis en forme à l'aide de balises prévus à cet effet, de nos jours il est plus judicieux d'utiliser le CSS et de n'utiliser le XHTML que pour le contenu.


L'avantage de l'utilisation d'un fichier CSS pour la mise en forme d'un site réside dans la possibilité de modifier tous les titres du site en une seule fois en modifiants une seule partie du fichier CSS. Sans ce fichier CSS, il serait nécessaire de modifier chaque titre de chaque page du site (difficilement envisageable pour les énormes sites de plusieurs milliers de pages).


D'autres points fort sont perceptible. Il est par exemple possible de créer une feuille de style spécifique pour l'impression des documents, ce qui permet de retirer tous les effets de style et toutes les parties inutile lors de l'impression. De même, une feuille de style peut être utilisée pour les utilisateurs d'un téléphone portable, ce qui permet de mieux gérer la mise en forme particulièrement pour les petits écrans de ces appareils.
